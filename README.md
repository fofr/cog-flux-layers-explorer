# Flux layers explorer

A Replicate model for exploring the effects of patching different layers of Flux Dev.

https://replicate.com/fofr/flux-dev-layers

This model uses:

- ComfyUI (checkout the workflow_api.json for the workflow)
- The blocks buster node for patching layers from the [ComfyUI_essentials](https://github.com/cubiq/ComfyUI_essentials) custom nodes, by [cubiq](https://github.com/cubiq)

## Examples

```
# All attention layers
attn=1.05

# All double blocks
double_blocks=1.05

# All single blocks
single_blocks=1.05

# All layers in double blocks 2
double_blocks.2=1.05

# All layers in double blocks 2, 3 and 4
double_blocks.[234]=0.9

# All img layers in double blocks 2
img=1.05
```
